# Awesome Vim Plugins

This is my personal collection of Vim plugins.

### [vim-plug](https://github.com/junegunn/vim-plug)

`:PlugInstall` to install the plugins
`:PlugUpdate` to install or update the plugins
`:PlugDiff` to review the changes from the last update
`:PlugClean` to remove plugins no longer in the list

### [tokyonight-vim](https://github.com/ghifarit53/tokyonight-vim)

A clean, dark vim theme that celebrates the lights of downtown Tokyo at night based on a VS Code theme with the same name.

### [lightline](https://github.com/itchyny/lightline.vim)

A light and configurable statusline/tabline plugin for Vim.
